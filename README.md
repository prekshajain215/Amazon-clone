# Amazon Clone

A frontend clone of Amazon's homepage built using HTML5 and CSS3.

## Features
- Responsive navigation bar
- Search bar with category selection
- Hero banner section
- Product showcase cards
- Footer section
- Clean and structured UI

## Technologies Used
- HTML5
- CSS3
- Flexbox

## Purpose
This project was built to strengthen my understanding of frontend web development, page layouts, responsive design, and user interface implementation.
